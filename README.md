# Venda Experience - Personal Shopper

A modern Next.js website for Venda Experience, a premium Takealot personal shopping service with a strong South African cultural identity.

## Features

- **Modern Next.js 14** with TypeScript
- **Tailwind CSS** for styling with custom South African color palette
- **Framer Motion** for smooth animations
- **FontAwesome Icons** for consistent iconography
- **Responsive Design** that works on all devices
- **South African Cultural Elements** including Ubuntu philosophy and local terminology

## South African Design Elements

### Color Palette
- **SA Gold** (#FFB612) - South African flag gold
- **SA Green** (#007749) - South African flag green  
- **SA Red** (#DE3831) - South African flag red
- **SA Blue** (#002395) - South African flag blue
- **Ubuntu Orange** (#FF6B35) - Warm, welcoming orange
- **Springbok Gold** (#F4B942) - Rich golden yellow

### Cultural Features
- Ubuntu philosophy integration ("I am because we are")
- South African greetings (Sawubona, Siyabonga)
- Local terminology (Mzansi, Venda, braai essentials)
- African pattern backgrounds
- Community-focused messaging

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Project Structure

```
├── app/
│   ├── globals.css          # Global styles with SA theme
│   ├── layout.tsx           # Root layout
│   └── page.tsx             # Main page
├── components/
│   ├── Navbar.tsx           # Navigation with SA branding
│   ├── Hero.tsx             # Hero section with animations
│   ├── Features.tsx         # Service features
│   ├── HowItWorks.tsx       # Process explanation
│   ├── Deals.tsx            # Exclusive deals section
│   ├── PaymentMethods.tsx   # SA payment options
│   └── Footer.tsx           # Footer with contact info
├── public/                  # Static assets
└── tailwind.config.js       # Tailwind with SA colors
```
